## 🎯 설명

VanilaJS를 활용한 간단한 웹페이지 구현

## 🚀 주요 스택

- Html
- css
- js
- weathermap(api)

## 🧩 구성

- 시작페이지
  - 이름을 입력하여 로그인 구현
- 배경
  - 랜덤으로 이미지 불러오기
- todo
  - 모달과 함께 todo 기능 구현
- 시계
  - 현재 날짜와 시간을 실시간으로 받아와서 매 초마다 갱신
- 위치/날씨
  - weathermap api를 통해 위치 및 날씨 정보 받아와 출력
- 명언
  - 드라마 명대사나 명언들을 랜덤으로 출력
